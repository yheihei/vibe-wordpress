Generating Pixel Art with AI: A Deep Dive

2024-07-20

In the world of retro game development, creating authentic pixel art is a crucial but time-consuming task. As an experiment, I delved into using Generative AI to speed up this process. This post documents my findings.

Choosing the Right Model

Not all image models are great at pixel art. I found that models specifically trained on or fine-tuned with pixel art datasets perform best. The key is using prompts that specify the style, like "8-bit sprite", "16-bit RPG character", "pixel art, side-scroller background".

Prompt Engineering for Sprites

For a character sprite, a good prompt might be: 'a pixel art sprite of a space knight, 32x32 resolution, limited color palette, NES style, white background'. The specificity is key to getting usable results.

The results were surprisingly good, although they often required manual touch-ups in a tool like Aseprite. It's not a complete replacement for an artist, but it's an incredible tool for brainstorming and generating base assets.
